# AI-impact-analysis
Анализ воздействия ИИ на профессии и зависимость риска увольнения и автоматизации
Источник данных
Данные взяты c общедоступного ресурса: (источник 1, источник 2)

Содержат данные о влиянии искусственного интеллекта на рабочие места и риск увольнений (1 датасет) и информацию о том, как автоматизация на основе искусственного интеллекта может повлиять на различные профессии, навыки и уровни образования к 2030 году (2 датасет)

Переменные и их описание:

1й датасет:

Age — Возраст работника

Education_Level — Уровень образования

Years_of_Experience — Стаж работы

Industry — Отрасль

Job_Role — Должность

Company_Size — Размер компании

Job_Level — Уровень должности

Routine_Task_Percentage — Процент рутинных задач

Creativity_Requirement — Требование креативности

Human_Interaction_Level — Уровень взаимодействия с людьми

AI_Adoption_Level — Уровень внедрения ИИ

Number_of_AI_Tools_Used — Количество используемых ИИ-инструментов

AI_Usage_Hours_Per_Week — Часы использования ИИ в неделю

Tasks_Automated_Percentage — Процент автоматизированных задач

AI_Training_Hours — Часы обучения ИИ

Layoff_Risk — Риск увольнения

2й датасет:

Average_Salary — Средняя годовая зарплата (в долларах)

Years_Experience — Требуемый стаж работы

Education_Level — Уровень образования

AI_Exposure_Index — Индекс подверженности автоматизации ИИ

Tech_Growth_Factor — Коэффициент технологического роста

Automation_Probability_2030 — Вероятность автоматизации к 2030 году

Risk_Category — Категория риска

Skill_1 - Skill_10 — уровень владения навыком

Данные очищены, удалены ненужные столбцы, дубликаты, пропуски заменены модой и медианой, датафреймы соединены

Переменные изучены по их типу (категориальные и числовые), для этого созданы 2 переменные для каждого типа. Результаты визуализированы

Изучено взаимодействие переменных: Числовая + категориальная: 'Layoff_Risk' + 'Automation_Probability_2030', Категориальная + категориальная: 'Layoff_Risk' + 'Education_Level', поставлены исследовательские вопросы, визуализированы результат
